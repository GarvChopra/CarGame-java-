# Car Game 🚗

A simple 2D car game built in Java using AWT/Swing. The player controls a car and avoids obstacles while the game continuously increases in difficulty.

---

## 🚀 Features

- 🎮 Player-controlled car movement  
- 🚧 Obstacle avoidance gameplay  
- ⚡ Real-time game loop  
- 💥 Collision detection  
- 🧠 Score / Game Over system  

---

## 🛠️ Tech Stack

- Java  
- AWT / Swing  

---

## ▶️ How to Run

```bash
# Compile
javac *.java

# Run
java main
